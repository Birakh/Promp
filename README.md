# Prompt Vault V5

A static, GitHub Pages-ready product that helps users turn rough ideas into stronger prompts for Claude, Copilot, ChatGPT, and Gemini.

## What V5 changes
This is no longer just a prompt library. It now works like a **raw-idea → better-prompt** system.

Users can:
- type a rough idea
- choose a task type
- get:
  - best default prompt
  - stronger version
  - fast version
  - model-fit note
  - what changed
  - why it works
- copy the result into their preferred AI tool
